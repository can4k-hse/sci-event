# Create PR

Создай коммит и PR для текущей ветки по следующему алгоритму:

1. Получи имя текущей ветки через `git branch --show-current`
2. Посмотри staged и unstaged изменения через `git diff HEAD` и `git status`
3. Посмотри коммиты ветки относительно main через `git log main..HEAD --oneline`
4. Сгенерируй краткое описание изменений на русском языке (1–2 предложения) — это будет {описание}
5. Создай коммит с сообщением в формате: `{ветка}: {описание}`
   - Пример: `FRONTFRAMEWORK-4: добавлены цветовые токены и CSS Modules для компонента Button`
   - В конце добавь Co-Authored-By трейлер
6. Запушь ветку в remote: `git push -u origin {ветка}`
7. Создай PR через `gh pr create` с:
   - title: `{ветка}: {описание}`
   - body: краткое описание что сделано и зачем (на русском)
8. Верни ссылку на PR
